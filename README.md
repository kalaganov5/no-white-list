Решение чтобы выходить в интернет, когда мобильный интернет режут/фильтруют по «белым спискам».

1. Скачиваем и устанавливаем приложение
   - [Google Play](https://play.google.com/store/apps/details?id=com.happproxy)
   - [AppStore](https://apps.apple.com/ru/app/happ-proxy-utility-plus/id6746188973)
   - [Windows](https://github.com/Happ-proxy/happ-desktop/releases/latest/download/setup-Happ.x64.exe)
   - [Mac](https://github.com/Happ-proxy/happ-desktop/releases/latest/download/Happ.macOS.universal.dmg)
   - [Другие платформы](https://www.happ.su/main/ru)

2. Копируем
   ```
   https://raw.githubusercontent.com/zieng2/wl/main/vless_lite.txt
   ```

3. Нажимаем + вверху справа

4. Выбираем добавить подписку
   - Включаем - разрешить небезопасные
   - Вставляем в URL - скопированный выше адрес

[🎥 Видео инструкция: Добавление подписки](https://itdomaster.ru/wp-content/uploads/2026/03/add-subcribe.mp4)

5. Желательно: Настроить обновление подписки

[🎥 Видео инструкция: Настройка обновление подписки](https://itdomaster.ru/wp-content/uploads/2026/03/update-subscibe.mp4)

## Как выбрать рабочий сервер

1. В клиенте запускаешь тест задержки/пинг для всей подписки.​
2. После проверки клиент покажет список серверов с задержкой (ping/latency).
3. Выбираешь сервер с наименьшим временем — как правило, он будет самым быстрым/стабильным.

Многие российские серверы в подписке уже дают доступ к заблокированным сервисам, поэтому специально подключаться к иностранным не обязательно.
